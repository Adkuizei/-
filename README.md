<p>抖音用户数据分析数据集</p>
<P>数据列名列表
Uid：用于唯一标识每个用户的字段。
User_city：记录每个用户所在城市。
Item_id：用于唯一标识每个作品的字段。
Author_id：用于唯一标识抖音作品的作者的字段。
Item_city：记录抖音作品所关联的城市。
channel：记录用户观看抖音来源的渠道。
Finish：记录了用户是否浏览完了整个抖音作品。
Like：记录了用户是否点赞作品。
Music_id：用于唯一标识抖音作品中使用的音乐。
Duration_time：记录了抖音作品的时长，单位为秒。
Real_time：记录了抖音作品发布的时间。
H：记录了用户观看抖音作品的小时数。
Data：记录了用户观看抖音作品的日期。</P>
